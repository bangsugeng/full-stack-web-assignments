1. Apa itu responsive web design?

Responsive adalah teknik yang digunakan pada pengembangan website agar pengguna dapat dengan mudah mengakses website pada device apapaun tanpa mengurangi estetika interface website [tetap rapi & tertata]

3. Apa itu breakpoint? Bagaimana cara kita membuat hal tersebut?

breakpoint adalah titik yang menjadi prasyarat style untuk dapat menyesuaikan ukuran desain/layoutnya biasanya disesuaikand dengan layar device atau bisa juga menggunakan breakpoint di developer tools pada browser komputer
cara membuatnya adalah dengan menambahkan code style @media only screen and (max-width: ...) {...}
