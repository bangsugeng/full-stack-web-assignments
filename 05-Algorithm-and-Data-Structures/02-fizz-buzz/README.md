### Tugas 02-fizz-buzz

```
# Start Writing Here

WHILE Nilai <= 20
    IF Nilai MOD 3 = 0 && MOD 5 = 0 THEN
        DISPLAY "FIZZBUZZ"

    ELSE IF Nilai MOD 3 = 0 THEN
        DISPLAY "FIZZ"

    ELSE IF Nilai MOD 5 = 0 THEN
        DISPLAY "BUZZ"
        ELSE
        DISPLAY Nilai

    END IF
    
    Nilai = Nilai+1
END WHILE
```
