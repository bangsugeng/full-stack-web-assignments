1. Apa itu CSS Framework?

CSS Framework adalah library kode CSS yang sudah siap pakai, CSS Framework ini biasanya digunakan oleh pengembang website untuk mengembangkan projectnya agar dapat dikerjakan lebih cepat karena tinggal copy-paste-edit

bootstrap dapat digunakan secara lokal maupun online
